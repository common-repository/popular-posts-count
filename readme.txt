Popular Posts Count is a widget plugin which displays popular posts by VIEWS / COMMENTS on your blog in cool style.




== Description ==

Display Post(s) or Products View Counts on Frontend and Backend without creating an Extra table in your database, hence providing results without any load to the server.

Display popular POSTS / PRODUCTS using Post / Product Views or Number of Comments for a post / product respectively.<br>


<strong>Main Features:</strong>
<ul>
<li>Display view counts without creating extra tables in your database.</li>
<li>Control, how many posts / products to show in widget</li>
<li>Multiple Skins</li>
<li>Display views on single POSTS / PRODUCTS using shortcode</li>
<li>Display views on single PAGE using shortcode</li>
<li>Also available with Dashboard Widget to view popular posts on admin dashboard</li>
</ul>

<strong>You can customize:</strong>
<ul>
<li>Widget’s title</li>
<li>Show popular posts from POSTS / PRODUCTS</li>
<li>Show popular posts by VIEWS / COMMENTS</li>
<li>How many posts to show</li>
<li>Show most popular posts from last 1, 2, 3, 7, 21, 30, 60 or 90 days</li>
<li>You can change the tag name eg.(views / comments)</li>
</ul>


== Installation ==


To install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Use the Dashboard->Popular Posts Count to configure the plugin



<br>
For Output / Result:<br>
For Post or Page or Products: <strong>[post-views] views/visits</strong><br>
For PHP: <strong>&lt;?php echo do_shortcode('[post-views] views/visits'); ?&gt;</strong>
